# 迹忆 (Remi) —— 万水千山，皆成迹忆

**迹忆 (Remi)** 是一款专为 **HarmonyOS Next** 打造的深度感官足迹记录应用。它不仅仅记录你“去过哪里”，更通过物理足迹与情感记忆的交织，帮你留住旅途中的气味、声音与心境。

> **Slogan:** 万水千山，皆成迹忆 (Where every mile becomes a memory)

---

## 🌟 产品愿景

在数字时代，足迹不应只是地图上的坐标，而应是鲜活的人生故事。**迹忆** 通过鸿蒙原生的分布式能力与感官化记录逻辑，将冰冷的数据转化为有温度的私人数字博物馆。

---

## ✨ 核心功能 (MVP 版本)

### 1. 动态涟漪地图 (The Ripple Map)

* **物理足迹可视化：** 采用原生 `Map Kit`，地点以动态“涟漪”形式呈现，而非生硬的图钉。
* **智能点亮：** 自动统计已点亮的国家、城市，并生成专属的旅行成就看板。

### 2. 五感标签记录 (Sensory Tags)

* **超越图文：** 支持记录嗅觉（如：雨后泥土香）、听觉（环境音采集）、味觉等感官标签。
* **环境音采集：** 利用 `Audio Kit` 捕捉当下的背景声，让记忆具有听觉深度。

### 3. 时光回响 (The Echo)

* **桌面万能卡片：** 通过鸿蒙元服务卡片，在桌面轮播“历史上的今天”，唤醒深藏的记忆。
* **智能问候：** 系统会基于过往记录，发起跨越时空的情感互动。

### 4. 鸿蒙原生体验 (HarmonyOS Next Native)

* **跨设备流转：** 手机记录，平板/折叠屏即时同步，大屏沉浸式回味。
* **隐私保护：** 严格遵循 HarmonyOS 最小权限原则，足迹数据本地加密存储。

## 🛠 技术栈

* **开发语言：** ArkTS
* **UI 框架：** ArkUI (声明式 UI 范式)
* **系统能力：** * **Map Kit：** 地图渲染与地理围栏。
* **Location Kit：** 高精度定位。
* **Media Library Kit：** 多媒体资产管理。
* **ArkData (RDB)：** 分布式关系型数据库。
* **Distributed Service Kit：** 跨端任务流转与数据同步。



---

## 🎨 设计哲学

* **Logo 寓意：** 以“足印”为核心，外围扩散出类似年轮或水波的“记忆涟漪”，象征经历随时间而沉淀。
* **字体：** 采用具有“手写旅行笔记”温度感的现代字体，强调个人化与独特性。
* **配色：** 清新的青绿（自然/探索）与阳光黄（高光时刻）交织。

## 🚀 快速开始 (开发者)

### 环境要求

* DevEco Studio Next Beta 及以上版本。
* HarmonyOS Next SDK (API 21+)。

### 安装与运行

1. **克隆仓库：**
```bash
git clone https://github.com/your-repo/remi-harmonyos.git

```


2. **打开项目：** 使用 DevEco Studio 打开项目文件夹。
3. **配置签名：** 在 `Project Structure` 中配置您的开发者签名。
4. **运行应用：** 连接 HarmonyOS Next 真机或模拟器，点击 `Run`。
